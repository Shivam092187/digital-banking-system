# 💳 Digital Banking System

A secure full-stack banking system built using the MERN stack that simulates real-world banking operations like account management, deposits, fund transfer, and transaction tracking using a ledger-based architecture.

---

## 🌐 Live Demo
👉 https://digital-banking-system-2.onrender.com

---

## 🚀 Features

- User Authentication (JWT)
- Create & Manage Bank Accounts
- Add Funds (Deposit)
- Money Transfer Between Accounts
- Ledger-Based Transaction System
- Transaction History Tracking
- Secure Backend APIs with Middleware Protection
- Email Service: Nodemailer

---

## 🛠 Tech Stack

Frontend: React.js, Tailwind CSS, Axios  
Backend: Node.js, Express.js  
Database: MongoDB, Mongoose  
Security: JWT Authentication, bcrypt
Email Notification System (Nodemailer Integration)

---

## ⚙️ Setup Instructions

Backend:
npm install  
npm start  

Frontend:
npm install  
npm run dev  

---

## 🧠 Key Concepts

- REST API Design  
- MVC Architecture  
- Ledger-Based Banking System  
- MongoDB Aggregation Pipeline  
- JWT Authentication  
- Secure Transaction Flow (DEBIT / CREDIT)
